Memory Maps now live right in the toolbar. Tap the cloud-of-dots icon and the panel opens just beneath it - no floating window to drag around, which feels especially good on a tablet. Tap the icon again, tap away, or press Escape to close it. At the top there's a new Live view switch: flip it on to keep the active map's dots glowing on the canvas while you draw, so you can see exactly what a connecting brush will reach for - and the toolbar icon lights up so you always know it's on. It's part of a bigger tidy-up: we're moving the floating panels into the toolbar so everything sits a tap away, wherever you're drawing.

- Open Memory Maps from a cloud-of-dots icon in the toolbar - no window to drag
- A new Live view switch keeps the active map's dots on the canvas as you draw
- The toolbar icon lights up while Live view is on
- Friendlier on tablets and touch: tap to open, tap away to dismiss
