A few tools and words got clearer names, so the app reads the way it works. The main brush - a round line plus its connecting web - is now simply called Web. The marks your drawing remembers are called dots everywhere. And the old Invisible brush is now Seed: it sows dots for the web to grow on, without leaving any ink on the layer.

- The main brush is now called **Web** - a round line plus its connecting web
- Your drawing's remembered marks are **dots** everywhere now, not "points"
- The Invisible brush is now **Seed** - it sows dots for the web and leaves the layer blank
- A friendlier undo message, so you always know exactly what came back
