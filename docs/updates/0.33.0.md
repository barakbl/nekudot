Erasing used to rub out only the ink - the invisible dots that pull your web stayed behind, so a new stroke would weave right back into the space you thought you'd cleared. Now the eraser forgets those dots too. A new Erase mode lets you choose what a wipe removes: the paint and the dots together (the default), just the paint, or just the dots. And if you go too far, one undo brings back both.

- Erasing now forgets the dots under it, so the web stops reaching into cleared areas
- A three-way **Erase** mode: Paint + dots (default), Paint only, or Dots only
- Dots only is a forget brush - thin out the memory without touching your art
- Undo brings back the paint *and* the dots together
