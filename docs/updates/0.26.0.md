There's a new web style to try: Chrome. As you draw, it threads your points with lines in random, darkened tones of your colour, so the web shimmers like brushed metal instead of sitting in one flat shade - a faithful nod to the chrome brush from mrdoob's Harmony. A new Blend dial sets how the lines catch the light: Lighten for a bright metallic sheen on the dark canvas, Screen for more of a glow, or Darken and Multiply for the classic look on a light background. Open the Web settings (press c), pick Chrome under Classic, and try a gradient or rainbow colour to tint the shimmer.

- Chrome: a new metallic web style, ported from Harmony's chrome brush
- Every line takes a random darker tone of your colour, for a shimmering, mottled sheen
- A Blend dial - Lighten, Screen, Normal, Darken, Multiply - controls how it catches the light
