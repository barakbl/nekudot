Drawing on a tablet with a stylus? Turn on Pen only draws and rest your hand freely - only the pen marks the canvas, so a resting palm or a stray finger leaves nothing behind. Two-finger pan, pinch and rotate keep working as always. It is off by default, so finger drawing still works; find it in App settings under Input on any touchscreen.
