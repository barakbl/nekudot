Set Nekudot up just the way you like, then take it with you. Export a single settings file with your app preferences, brush presets and colour palettes, and import it on another device - or after a reset - to pick up right where you left off.

- Export everything - settings, custom presets and palettes - to one .nekudotapp file
- Import it on another device or browser to clone your setup in moments
- A tidy way to back up your preferences before a fresh start
