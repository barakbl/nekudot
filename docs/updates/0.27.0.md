The Memory Maps panel now opens with a plain-language note pinned at the top, so it is always clear what it is for. Every point your brush drops is remembered there, and the connecting webs weave lines between nearby points - now the panel says so, right where you need it, with no help mode to switch on.

- A short explainer stays pinned at the top of the Memory Maps panel
- Always visible - no help toggle to find
- Makes it clear your points are remembered, and webs connect the nearby ones
