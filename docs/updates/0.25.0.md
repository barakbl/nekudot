The Maps button in the toolbar is simpler now. Tap the highlight to light up your live point cloud on the canvas - a quick note tells you how many points you've laid down so far. Open the little chevron any time for that count, plus a shortcut straight into your Memory map settings.

- Tap the highlight to show your cloud points live on the canvas
- A chip tells you how many points you have right now
- The chevron opens the point count and a shortcut to Memory map settings
