Eraser

Rubs out pixels - without forgetting anything.

What it is

The eraser is a tool toggle, not a brush. Click the eraser button in the toolbar to turn it on; click it again (or pick any brush) to turn it off. While it's on, your strokes remove paint instead of adding it, clearing whatever is underneath on the active layer.

How it works with brushes

The eraser borrows the currently selected brush's shape and size - it doesn't have its own. A fat round brush erases in fat round dabs; a Squares brush erases square chunks; making the brush bigger makes the eraser bigger. It simply puts the current brush into "remove" mode, so everything you know about the brush still applies.

  • Pick the brush whose footprint you want, then toggle the eraser on.
  • It erases the active layer only - including any connecting web woven onto it.
  • Toggle the eraser off to go straight back to painting with that same brush.

The eraser and memory

This is the important part: erasing is purely visual. It clears pixels on the canvas but does not touch your drawing's memory. While erasing, nothing is added to or removed from that memory - the remembered points behind the paint stay exactly where they were.

So if you erase a stroke and then draw with a connecting brush, its web will still reach toward the points you "erased" - they're invisible but not forgotten. To actually forget points (and stop them attracting connections), clear the memory map instead - that's a separate action from erasing.
Check the memory Erase a stroke, then press m and Flash the map - you'll see its points glowing right where the paint used to be. That's why a new web still reaches there. It's a handy feature, not a bug: rub out the ink but keep the structure.

When to use which

  • Eraser - remove paint you don't want, while keeping the structure the web is built on.
  • Clear the map - forget the remembered points so new connections start fresh.
  • Layer opacity / delete - fade or drop a whole layer at once (see Layers).
