Marker key 8

A chisel-nib marker for solid, flat strokes.

What it does

Marker draws with an angled chisel tip, giving calligraphic strokes that are thick across and thin along the diagonal. By default it doesn't connect - it behaves like a plain marker, with no web, so it's your go-to for clean lettering and flat fills.

How to use

  • Press 8 or pick it from the brush menu.
  • Its opacity comes from the Opacity slider at the top of the brush settings panel (selecting Marker sets it to ~75%). Lower it to build up translucent layers.
  • Change the brush Size (also at the top of the settings panel) to widen the nib.
Try this Lower the Opacity and let strokes cross - each overlap stacks into a richer tone, like real marker ink. Want it to weave a web like the other brushes? Switch the Connecting style from No connect to Airy in the navbar combo.

Settings

Opacity
Uses the global Opacity slider at the top of the brush settings panel (shared by every brush).
Chisel follows pen
With a stylus, the chisel nib rotates to match the direction the pen leans - true calligraphy-marker behaviour. On by default; a mouse (or a perfectly vertical pen) keeps the classic 45° nib. See Pen & pressure.
Pen bindings
Pressure and tilt can drive the nib's width and opacity - the Pen section in this brush's settings. Full guide.
Connecting style
Off by default. Switch it from No connect to a style like Airy if you want the marker to weave a web too - see connecting.
