▭Squares key 2

Drops squares along your stroke, sized by how fast you move.

What it does

As you drag, Squares places rotated squares spaced along the path. The faster you move, the bigger each square (up to a cap) - so a quick flick leaves big bold tiles and a slow drag leaves a tight trail of small ones. It can also connect like the other brushes.

How to use

  • Press 2 or pick it from the brush menu.
  • Vary your speed to vary the size - that's the main expressive control.
  • Set Fill to none for outlines, or main/secondary for solid tiles.
Try this Flick fast for big bold tiles, then trail slowly for a tight run of small ones - one gesture can swell and taper like a brushstroke. Turn on a Connecting style to thread the square centres into a faceted, crystalline web.

Settings

Fill
None (outline only), Main, or Secondary color.
Fill opacity
Opacity of the filled square.
Dash
Outline style: solid, dashed or dotted.
Connecting style
Optional web between square centers - see connecting.
