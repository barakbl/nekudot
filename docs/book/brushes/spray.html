Spray key 4

An airbrush - soft specks that build up the longer you dwell.

What it does

Spray scatters specks within a radius around the pointer. The longer you hold still, the denser it builds (it keeps spraying while the pointer is down), and moving spreads a thinner trail. Each speck composites on its own, so overlapping specks stack into soft tone - the classic airbrush feel.

It also quietly feeds the drawing's memory: it drops points into the map as you spray (throttled, so it won't flood), which means you can spray a cloud and then switch to Round to weave a web over it.

How to use

  • Press 4 or pick it from the brush menu.
  • Dwell to build density; drag to lay a softer trail.
  • Selecting Spray drops the Opacity low (~20%) so it builds gently - raise it for a harder edge.
  • With a stylus, pressure drives density and spread (heavier = more, wider).
Try this Spray a soft cloud, then switch to Round and scribble over it - the web pulls toward the sprayed points. Pair that with a low Links and the Rainbow colour for an iridescent, nebula-like mesh.

Settings

Radius
How wide the spray scatters. Small = a tight stipple; large = a soft cloud.
Flow
How many specks land each frame. Higher = denser, faster build-up.
Dot size
Size of each speck. Tiny = fine grain; larger = a coarse, splattery spray.
Opacity
The global Opacity slider sets how strongly each speck marks - low values are what make the build-up soft.
Pen
With a stylus, pressure scales the density and spread (respects the pen toggle in Application settings).
