Symmetry panel key y

Pick a symmetry mode and shape how your strokes repeat.

What it is

A mode picker - None, Tile, Radial, Mirror - plus the parameters for the chosen mode and the on-canvas guide lines. Whatever brush is active gets repeated. Open it from the navbar Symmetry pill, the Windows menu, or y. For what each mode does, see Symmetry.

Tile parameters

X / Y spacing
The lattice pitch - distance between tile junctions.
Fill canvas
Tile the whole canvas at full strength (Reach and Falloff grey out). Best with a small motif.
Reach
How far the tiling spreads out from where you started.
Falloff
How sharply copies fade toward the reach edge (0 = even).

Radial & Mirror

Segments (Radial)
How many wedges the kaleidoscope splits into.
Mirror (Radial)
Reflect each wedge as well as rotating it.
Axis (Mirror)
Reflect across a vertical or horizontal line through the centre.

Guides

Opacity / Width / Color
Style the on-canvas guide lines (the lattice, spokes or mirror line) that show where copies fall. Guides sit above the drawing but below the panels.
