Toolbar

The floating bar at the top - every tool and quick action, left to right.

What it is

A single draggable bar that floats over the canvas - grab the dotted handle on its left to move it anywhere. Most controls are pills (a quick value plus a ⌄ dropdown that opens the matching panel). On a phone the bar wraps onto a few rows and trims itself: the brush pill shows only its icon, and undo/redo disappear (use the two- and three-finger tap gestures instead).

What each control does

Drag handle (⠿)
Drag to reposition the whole toolbar.
More (…)
Share / export PNG, save/load .nekudot, and Record GIF. (Theme and pen pressure moved to Application settings.)
Windows
Show any panel (Brush settings, Layers, Maps, Symmetry, Application settings, Shortcuts) with its keyboard shortcut listed.
Memory Maps pill
A flash button and a ⌄ dropdown. Click the flash to toggle the active map's live cloud-point highlight on the canvas (a chip shows the current point count); it stays lit while on. The dropdown shows the live point count and a Memory map settings row that opens the Maps panel.
Colour swatch
Front and back colours; right-click to swap them. Click one to open the Colours palette (palettes by category, a picker, import/export).
Brush pill
The selected brush (icon + name). The dropdown picks a brush; the connecting brush is shown as Web, with the connection styles (Classic / More / Custom) nested under it - pick one to select Web and apply that look, and the pill then shows the active style's name. The settings… row (or b) opens Brush settings; for Web it lands on the Web tab, where the style picker and its dials live.
Symmetry pill
The current symmetry mode (None / Tile / Radial / Mirror); the gear opens the Symmetry panel.
Undo / Redo
Step history back and forward (also ⌘/Ctrl Z, or 2/3-finger taps).
Delete canvas
Wipe all layers back to one (asks first).
New art
Start a fresh canvas at a size you pick.
Reset view
Return the canvas to its normal view - no zoom, no rotation, centred (or fit if it is larger than the window). See Navigating the canvas.
Full screen
Fill the whole display, hiding the browser's own bars; press again (or Esc) to come back. Shown only where the browser supports it - iPad Safari and desktop, not iPhone. See Install & full screen.
Tip Every pill's gear opens the same panel you'd reach from the Windows menu or its keyboard shortcut - see Shortcuts & gestures.
