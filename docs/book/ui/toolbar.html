Toolbar

The floating bar at the top - every tool and quick action, left to right.

What it is

A single draggable bar that floats over the canvas - grab the dotted handle on its left to move it anywhere. Most controls are pills (a quick value plus a gear that opens the matching panel and a ⌄ dropdown). On a phone the bar wraps onto a few rows and trims itself: the brush and connecting pills show only their icon, the maps pill drops its point count, and undo/redo disappear (use the two- and three-finger tap gestures instead).

What each control does

Drag handle (⠿)
Drag to reposition the whole toolbar.
More (…)
Theme (auto/light/dark), pen-pressure toggle, share/export PNG, save/load .nekudot, and Record GIF.
Windows
Show any panel (Brush settings, Connecting, Layers, Maps, Symmetry, Shortcuts) with its keyboard shortcut listed.
Memory Maps pill
The active map's live point count - click it to open the Maps panel; the target button flashes that map's dots on the canvas.
Colour swatch
Front and back colours; right-click to swap them.
Brush pill
The selected brush (icon + name). The dropdown picks a brush; the gear opens Brush settings.
Connecting pill
The connection art style (Classic, Web, Shading…). Only shown for brushes that weave a web; the gear opens the Connecting tab.
Symmetry pill
The current symmetry mode (None / Tile / Radial / Mirror); the gear opens the Symmetry panel.
Undo / Redo
Step history back and forward (also ⌘/Ctrl Z, or 2/3-finger taps).
Delete canvas
Wipe all layers back to one (asks first).
New art
Start a fresh canvas at a size you pick.
Reset view
Return the canvas to its normal view - no zoom, no rotation, centred (or fit if it is larger than the window). See Navigating the canvas.
Tip Every pill's gear opens the same panel you'd reach from the Windows menu or its keyboard shortcut - see Shortcuts & gestures.
