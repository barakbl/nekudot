Install & full screen

Nekudot runs in the browser, but you can add it to your home screen and run it full screen so it behaves like a native app - a bigger, calmer canvas with none of the browser's own bars in the way. Especially nice on an iPad.

Add it to your home screen

Installed, Nekudot opens in its own window - no address bar, no tabs - straight onto the canvas, with its icon and name on your home screen or dock.

iPad & iPhone (Safari)
Tap the Share button, then Add to Home Screen. Launch it from the new icon and it fills the screen on its own.
Desktop (Chrome / Edge)
Click the install icon in the address bar, or open the browser menu and choose Install Nekudot. It opens in a standalone window you can keep in your dock or taskbar.
It's the same app either way - it still runs entirely on your device and nothing is uploaded. See Save to a folder for keeping your work as real files (and syncing it across devices yourself).

Full screen

The Full screen button in the toolbar - the corner-brackets icon next to Reset view - hides the browser's own bars so the canvas fills the whole display. Press it again, or Esc, to come back; the icon flips to show which it will do next.

The button appears only where the browser can go full screen - iPad Safari and desktop browsers. iPhone Safari has no web full-screen mode, so the button isn't shown there; add Nekudot to your home screen instead for the same chrome-free, app-like canvas.
Tip On an iPad, adding Nekudot to the home screen already runs it chrome-free, so you may not need the full-screen button at all - it earns its keep most in a normal desktop browser tab.
