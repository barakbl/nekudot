Memory Maps panel key m

The named point clouds your connecting brushes reach toward.

What it is

A memory map is a set of remembered points - separate from the painted pixels and shared across layers. Connecting brushes weave their web toward these points, so a map lets a drawing "remember" what to link to. Open the panel from the navbar maps pill, the Windows menu, or m. For how maps feed the web, see Memory maps.

Controls

Active map
The map new points are deposited into. Click a row to switch; its live point count shows here and in the navbar pill.
Flash
The target button briefly lights up a map's dots on the canvas so you can see where they are.
Add / Rename / Delete
Make a new map, double-click to rename, or remove one.
Tip Keep a clean map of just the points you want to connect, then draw on a fresh layer that reaches into it - the web threads toward your remembered shape without re-drawing it.
