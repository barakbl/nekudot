Brush & Connecting settings key b

Tune the active brush and, for connecting brushes, the look of the web.

What it is

One panel with two tabs, opened from the brush pill's gear, the Windows menu, or b. The Brush tab holds size, opacity and the selected brush's own options; the Connecting tab (shown only for brushes that weave a web) holds the routing and the art-style dials. Reset reverts both to defaults.

Brush tab

Size
Stroke width.
Opacity
How solid the mark is. A faint continuous stroke composites at one even alpha (no darkening where caps overlap).
Brush options
Whatever the active brush exposes - e.g. the dash style, or shape sizing. See Brushes.
Pen
With a stylus, map pressure and tilt to size, opacity, even the web - see Pen & pressure. (Gated by the pen toggle in the More menu.)

Connecting tab

Connection
The art style (Classic, Web, Arc, Shading, Fur, Lace) or No connect. The quick picker also lives in the navbar Connecting pill.
Routing - Connect to stroke or map?
Where the web reads its points from and writes them to (the current stroke, a memory map, or both).
Art-style dials
Weight, Spread, Opacity, Density, Reach, Fade and more - each style surfaces the dials that shape it. Full reference in Connections.
Save as preset
Store the current dials as a reusable custom preset (managed from the navbar Connecting dropdown).
The tab bar only shows Connecting for brushes that connect (Round). Non-connecting brushes show just the Brush tab.
