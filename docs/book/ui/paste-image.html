Paste an image

Drop a picture from your clipboard straight onto the canvas, then position it before it is painted in.

What it is

Copy any image - a screenshot, a photo, a reference from the web - and press ⌘/Ctrl V in Nekudot. The image appears as a movable preview on the canvas; once it is where you want it, Place bakes it onto the active layer. Until then nothing is committed, and the bake is a single undo step.

The placement preview

On paste, the image is centred on the canvas and, if it is bigger than the canvas, scaled down to fit. A dashed box with four corner handles shows where it will land:

Move
Drag anywhere inside the box.
Resize
Drag a corner handle. The aspect ratio is kept, anchored to the opposite corner.
Place
Click Place (or press Enter) to bake the image onto the active layer.
Cancel
Click Cancel (or press Esc) to drop it with no change.

The preview rides the canvas, so panning, zooming and rotating the view all keep it glued in place while the handles stay a constant size on screen.

It lands on the active layer. The image is painted into whichever layer is selected, at full opacity. Pick or add a layer first if you want it on its own.
Try this Paste a reference photo onto its own layer so you can draw on a layer above it, then hide or delete the reference when you are done.
