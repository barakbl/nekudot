Colours palette

A floating colour picker that pops up next to the swatch you opened it from - browse palettes by category, mix a colour, and pick it.

What it is

Click a colour swatch - the toolbar's front/back swatches, or the background swatch in the Layers panel - and the palette popover opens beside it, like the OS colour picker. Picking a colour applies it and closes the popover; clicking outside or pressing Esc also closes it. Everything you do here - your palettes, recent colours, the chosen tab and category - is saved locally on this device.

Layout

Recent
The colours you picked most recently, newest first - one click to reuse one. Clear empties the row.
Palette tab
All your palettes as rows of swatches. A Category dropdown filters them; the palette you last picked from floats to the top.
Picker tab
Mix an exact colour: OKLCH (perceptual lightness / chroma / hue sliders) or HSB (a saturation-brightness square + hue bar), a hex field, an eyedropper where the browser supports it, and Apply.

Categories

Every palette carries one category, and the Palette tab's dropdown filters by it. They mix feels and themes - Calm, Hot, Cool, Vibrant, Earthy, Pastel, Animals, Fruits and General (the default for a palette you haven't tagged). Choose All categories to see everything at once.

Edit mode

The popover opens in a clean pick-only view. Tap Edit (top-right of the popover) to reveal the editing tools; tap Done to hide them again. While editing you get:

+ New palette
Name a palette, give it a category, then add swatches and pick each colour. It's saved alongside the rest.
Import
Opens the bundled palettes you don't already have, grouped by category in collapsible sections (all closed; tap a category to reveal its palettes), plus an Upload .gpl file button - see below.
Export JSON / Import JSON
Back up your palettes: Export JSON opens a checklist (all ticked) and downloads the chosen palettes as one file with colours in OKLCH; Import JSON reads such a file back, merging by palette.
Per-palette actions
Each palette row gains edit (rename, retag, change colours), export and delete icons, plus a Gradient toggle.

Import & export

Nekudot reads and writes GIMP palette files (.gpl) - the plain-text colour-list format that GIMP, Krita, Inkscape and Aseprite all export.

Import a bundled palette
The Import dialog lists every palette that ships with the app but isn't in your list yet. Tap one to add it. (Delete a palette and it becomes available to add again - the originals are never lost.)
Upload a .gpl
Pick a .gpl file from your computer and its colours arrive as a new palette.
Export
The export icon on a palette downloads it as a .gpl you can open in GIMP and friends - or re-import here.

Gradients

Turning on a palette's Gradient toggle makes it a colour source for the connecting lines: it then shows up in the connecting Color dial, where the line colour sweeps through the palette around the circle. The default gradient palettes (Ocean, Sunset, Fire, Neon, …) ship with this on.

First run A starter set of palettes - the app colours plus category-tagged gradients - is added automatically the first time you open the app (and again if you reset to default). They're ordinary palettes after that: edit, retag or delete them freely.
