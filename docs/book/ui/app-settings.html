Application settings key ,

App-wide settings in one place - theme, input and advanced toggles.

What it is

The global counterpart to the per-brush Brush settings panel. Open it from the toolbar Windows menu or press ,. Theme and Pen pressure used to live in the toolbar's "…" menu and now live here.

Settings

Theme
Auto follows your system, or pin Light / Dark.
Pen pressure
Use a stylus's pressure and tilt to shape the stroke (and show the Pen section in Brush settings). Off makes a pen draw like a mouse - see Pen & pressure.
Pixel log
Records every deposited point to an append-only log. Off by default and best left off - it is groundwork for future features, nothing uses it yet, and it only grows stored data.
Reset to default
Under Reset: permanently erases every setting, layer and saved artwork on this device, then reloads the app to a fresh Start page. You must type yes to confirm - there is no undo.
These are application preferences, saved on your device. For settings that belong to the active brush (size, opacity, connecting dials), use the Brush settings panel instead.
