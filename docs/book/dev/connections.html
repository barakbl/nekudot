Writing a connection

A connection (web, fur, lace…) is one ConnectionBase subclass plus a one-line row in connections.json.

What a connection is

The connecting half of the app lives in src/brushes/connections/. ConnectionBase owns all connecting state and the engine that weaves the web - finding neighbours, drawing the lines, and fanning each one into Weight×1px hairs. A connection style is a subclass that just declares its defaults, the extra dials it exposes, and (optionally) a custom texture hook. It then shows up automatically in the navbar Connecting combo, and any connecting brush (Round) can wear it.

1. A minimal connection

Default-export a ConnectionBase subclass and name its menu glyph. defaults() returns the full flat set of connecting values (the same keys the settings panel shows) - everything the engine needs to draw. 

// src/brushes/connections/hatch.ts
import { ConnectionBase } from "./base";
import type { ConnectingFlat } from "../../connecting-types";

export const icon = "…svg markup or a single char…";

export default class HatchConnection extends ConnectionBase {
  protected defaults(): ConnectingFlat {
    return {
      alpha: 0.12, color: "main", connect: "line", dash: "solid",
      density: 40, radius: 50, minDist: 0, inset: 0, fade: 0,
      strands: 1, spread: 6, scatter: 0, taper: 0, flow: 0, fray: 0,
      length: 1, wave: 0, dynamics: 0, curl: 0,
      grainStrength: 0.85, grainAngle: 45, grainCross: false, // comb at 45°
    };
  }
}

Routing (which map a stroke reads/stores, and whether it connects) keeps the shared defaults - a connection only sets art style. See Memory maps & connecting for what each key does.

2. Expose your own dials

Every connection shows the universal dials (Weight, Spread, Opacity, Density, Reach, Grain, Line shape, Dash, Color…). Override extraSliders() to surface the ones specific to your style - use the numStyle() helper so they read the live value. 

protected extraSliders(): BrushSetting[] {
  return [
    this.numStyle("taper", "Taper", 0, 1, 0.05),
    this.numStyle("wave", "Wave", 0, 1, 0.05),
  ];
}

A dial is supported only if it's in baseSliders() or your extraSliders(); the panel shows the ones a style actually uses by default and folds the rest under “More.”

3. A custom texture

For multi-strand styles, each connection fans into hairs via drawFanned(), which calls drawHair() per hair. The base draws a straight 1px line; override drawHair() to shape it - exactly how FurConnection gets its tapering, combed, wavy pelt. Draw with this.drawConnection() and pull randomness from this.random() (the brush's seeded RNG, so a redraw is reproducible). 

protected drawHair(
  rx: number, ry: number, tx: number, ty: number,
  px: number, py: number, style: LineStyle,
  taper: number, flow: number, wave: number, phase: number,
): void {
  // …split the root→tip segment into a curved, fading polyline…
  this.drawConnection({ id: 0, x: rx, y: ry }, { id: 0, x: tx, y: ty }, style);
}

4. Open dials by default

Override defaultOpenKeys() to add your signature dials to the set the panel always shows (the base already opens Weight, Spread, Opacity, Density, Reach, Dash & Color): 

defaultOpenKeys(): readonly string[] {
  return [...super.defaultOpenKeys(), "grainStrength", "grainAngle"];
}

5. Register the connection

connections.json is grouped - "Classic", "More" and "Custom" (empty; user presets are injected at runtime). Add one row to a group array (its position is the combo order). The grouped combo, icons and createConnection() are generated from it via import.meta.glob - nothing else to touch. strokeAlpha is optional: the owning brush's stroke-line opacity for this style (Harmony's sketchy 0.05 / web 0.5 / shaded 0). 

// src/brushes/connections/connections.json
{
  "More": [
    { "name": "hatch", "file": "hatch.ts", "info": "Combed straight hatching.",
      "strokeAlpha": 0.1 }
  ]
}

A data-only style (Airy, Arc, Lace…) skips the subclass entirely: its row carries "file": "classic.ts" + a "defaults" object and runs on the generic class. Custom presets the artist saves are just rows persisted to IndexedDB (custom-store.ts) and merged into the registry via setCustomPresets().

Hooks on ConnectionBase

defaults(): ConnectingFlat
Required. The full flat set of connecting values your style starts from.
extraSliders(): BrushSetting[]
The dials this style supports beyond the universal ones (use numStyle()).
defaultOpenKeys(): string[]
Style keys the panel shows open by default, regardless of value.
drawHair(...)
Shape one fanned hair (override for fur-like textures); the base draws it straight.
drawConnection(p1, p2, style) / random()
Draw a line onto the connection layer; pull seeded randomness.
Adding a connecting brush instead? See Writing a brush - a brush opts into connecting with this.initConnection("classic").
