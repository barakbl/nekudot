Shaded Soft Pencil

Soft, build-up shading — like a pencil rubbed across paper.

What it does

Soft Pencil lays down no solid line. Instead, each point of your stroke draws faint lines to nearby points, and — the key part — each line's opacity fades with distance: close points draw darker, far ones fade to nothing. Scribble back and forth and the overlapping faint lines accumulate into a smooth, soft tone.

It's a port of mrdoob's Harmony “shaded” brush (harmony) — full credit for the idea is his. In Nekudot it's built on the standard connecting machinery with the Alpha falloff control turned up.

How to use

  • Press 3 or pick it from the brush menu.
  • Scribble over an area; pass over it again to deepen the tone.
  • Tune Radius for how far it reaches, Density % for how solid the fill is, and Alpha falloff for how fast lines fade with length.

vs. Web

Web draws sparse, equal-opacity strands for structure. Soft Pencil draws dense, short lines whose opacity fades with distance for tone — the Alpha falloff setting is what separates the two. See Memory maps & connecting for the full list of connection controls.
