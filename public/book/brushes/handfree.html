Handfree key 7

A freehand round-with-connections brush whose every mark is mirrored across a grid - draw once, and the motif tiles into a pattern.

What it does

Handfree is a Round brush that mirrors whatever you draw across a grid. The little motif you draw snaps to the grid junction nearest where you start, and every line and connection in it is copied to each junction within reach, fading toward the edge. A single dot becomes a field of dots; a richer flourish tiles into wallpaper. Because it's a connecting brush, it also wears any connection style.

How to use

  • Press 7 or pick it from the brush menu.
  • Draw a small motif near one grid junction; it copies to every junction in reach.
  • Set the pattern density with X / Y spacing, and how far it tiles with Reach.
  • Pick a connection style from the navbar Connecting combo to texture the tiled web.

Settings

X spacing / Y spacing
Distance between grid junctions horizontally / vertically - the pattern's pitch.
Reach
How far the motif tiles out from the anchor junction; copies fade toward the edge.
Opacity falloff
How sharply the copies fade with distance. 0 = flat (every copy full strength); higher = steeper fade.
Dash & connection dials
Shared with Round - see Connections and Memory maps & connecting.
Try this Draw one small flower or star near a junction and watch it bloom into a whole repeating pattern. Tighten X / Y spacing for dense wallpaper, widen it for an airy lattice, and add a Connecting style to thread the tiles together.
