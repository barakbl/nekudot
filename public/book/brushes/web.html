Web

The same round mechanism, tuned for crisp spider webs.

What it does

Web shares Classic Round's engine but selects the Web art style: sparser, slightly stronger, full-length lines (no inset). Instead of soft shading you get distinct strands stretched between points - like a cobweb catching your gesture.

How to use

  • Press 2 or pick it from the brush menu.
  • Selecting it applies the Web connection art style and a medium stroke opacity.
  • Move across existing marks to sling strands toward them.
  • Lower Density for a delicate web; raise Radius to reach across gaps.

Settings

Dash
Line style for the stroke itself.
Connection & Connection art style
Routing and look of the web - see Memory maps & connecting. Try the Arc art-style preset for curved strands.
